# FlashChat iOS-app
 
Проект с курса **"iOS & Swift - The Complete iOS App Development Bootcamp"**. 

Приложение-мессенджер с регистрацией и авторизацией с базой данных, хранящей пользователей и сообщения. 

## Used technologies 
- Cocoapods
- UIKit
- FirebaseAuth
- Firestore database
